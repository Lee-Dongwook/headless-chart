# 소개

**headless-chart**는 [Flitter](https://github.com/meursyphus/flitter) 프레임워크를 기반으로 구축된 고도로 커스터마이징 가능한 차트 라이브러리입니다. 이는 차트를 정의하고 구성하는 강력한 **헤드리스** 방식을 제공합니다. 기존의 차트 라이브러리처럼 미리 정의된 컴포넌트에 제약받지 않고, Flitter 위젯 형태의 빌딩 블록을 제공하여 원하는 대로 모양과 스타일을 지정할 수 있습니다.

## 주요 기능

- **완벽한 커스터마이징:** 차트 옵션을 조정하는 대신, 위젯 구조를 직접 수정할 수 있습니다.
- **Flitter 기반 레이아웃:** Flitter로부터 자동 레이아웃 계산, SVG/Canvas 듀얼 렌더러 지원, 내장된 인터랙션과 애니메이션을 상속받습니다.
- **확장성과 유연성:** 커스텀 위젯 함수를 구현하여 자신만의 막대, 범례, 축, 그리드를 구성할 수 있습니다.

즉, headless-chart는 Flitter의 위젯 시스템 위에 차트의 "뼈대"를 제공하여 최소한의 제약으로 원하는 유형의 차트를 구축할 수 있게 합니다.

## 학습 내용

이 가이드를 통해 다음을 배우게 됩니다:

1. 프로젝트에 headless-chart를 설치하고 설정하는 방법
2. Flitter와 통합하고 순수 Vite 환경에서 차트를 실행하는 방법
3. React 앱에서 렌더링을 단순화하기 위해 `@meursyphus/flitter-react`를 사용하는 방법
4. `BarChartCustom` 설정을 사용하여 차트 컴포넌트를 커스터마이징하는 기본 개념

## 사전 요구사항

- **Flitter 기본 지식:** headless-chart가 Flitter와 원활하게 작동하지만, Flitter의 위젯 시스템에 대한 기본적인 이해가 도움이 됩니다.
- **React (선택사항):** React와 통합하고자 하는 경우 React에 대한 기본적인 이해가 도움이 되지만, 필수는 아닙니다.

이제 headless-chart를 설치하고 환경을 설정해보겠습니다.
